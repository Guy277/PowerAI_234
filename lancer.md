# 🚀 Lancer l'application PowerBank

1. **Installer les dépendances (si nécessaire) :**
```bash
pip install streamlit pandas numpy scipy matplotlib seaborn faker plotly
```

2. **Lancer l'application :**
```bash
streamlit run main.py
```

---
*💡 Commentaire : Ne lancez pas le fichier directement avec `python main.py` car les fonctions d'interface web dépendent du serveur Streamlit. Saisissez la commande ci-dessus dans votre terminal, puis ouvrez [http://localhost:8501](http://localhost:8501) dans votre navigateur.*
